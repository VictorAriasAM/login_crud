# Login CRUD

Sistema web desarrollado con PHP y MySQL para la gestión de productos y proveedores mediante operaciones CRUD.

## Tecnologías utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Inicio de sesión de usuarios
- Gestión de productos
- Gestión de proveedores
- Alta de registros
- Consulta de registros
- Actualización de registros
- Eliminación de registros
